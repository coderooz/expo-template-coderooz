# Quick Start

## 1️⃣ Install the Template

```bash
npx create-expo-app -t expo-template-coderooz
cd <your-app>
npm start
````

## 2️⃣ File Structure

```
src/
 ├─ components/
 ├─ screens/
 ├─ context/
 ├─ hooks/
 ├─ theme/
 ├─ constants/
 └─ lib/
```

## 3️⃣ Environment Setup

* Android Studio installed
* Expo Go OR Dev Client
